# 🎮 Tic Tac Toe Game

This is my **first GitHub project**! 🚀

A simple Tic Tac Toe game built to practice programming fundamentals, logic building, and basic game development.

## 📌 Features

- 🎲 Two-player gameplay
- ✅ Detects win, draw, and ongoing game
- 🔄 Restart Game option
- 🎨 Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript


## 🚀 How to Run

1. Copy this link:
   ```
   https://dev-rohon.github.io/Tic-Tac-Toe/
   ```

2. Open chrome and paste the url.

3. enjoy the game.
